🚀 Cursor2API · Go Compatibility Build

这是当前 Go 版的真实能力页，不再是旧时代的虚假战报。当前版本已支持 OpenAI Chat Completions、Anthropic Messages、OpenAI Responses、tools / function calling、 refusal 拦截、identity probe mock、auto-continue，以及 vision / OCR 预处理。

服务地址

http://localhost:8002

默认 API Key

0000

主力模型

claude-sonnet-4.6

当前状态

✅ Streaming / Tools / Vision 已接通

能力摘要

OpenAI Chat Completions Anthropic Messages OpenAI Responses Tools / Function Calling SSE Streaming Refusal Retry / Sanitize Identity Probe Mock Vision / OCR

主要端点

GET/v1/models

返回可用模型列表。

curl -H "Authorization: Bearer 0000" http://localhost:8002/v1/models
POST/v1/chat/completions

OpenAI Chat Completions 兼容接口，支持流式、非流式、tool_calls。

curl -X POST http://localhost:8002/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer 0000" \
  -d '{
    "model": "claude-sonnet-4.6",
    "messages": [{"role": "user", "content": "Hello!"}],
    "stream": false
  }'
POST/v1/messages

Anthropic Messages API 兼容接口，支持 `content_block_*` / `message_*` SSE。

curl -X POST http://localhost:8002/v1/messages \
  -H "Content-Type: application/json" \
  -H "x-api-key: 0000" \
  -d '{
    "model": "claude-sonnet-4.6",
    "max_tokens": 1024,
    "messages": [{"role": "user", "content": "hello"}]
  }'
POST/v1/messages/count_tokens

轻量 token 估算接口。

curl -X POST http://localhost:8002/v1/messages/count_tokens \
  -H "Content-Type: application/json" \
  -H "x-api-key: 0000" \
  -d '{
    "model": "claude-sonnet-4.6",
    "messages": [{"role": "user", "content": "hello world"}]
  }'
POST/v1/responses

OpenAI Responses API 兼容接口，适配 Cursor IDE Agent 模式。

curl -X POST http://localhost:8002/v1/responses \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer 0000" \
  -d '{
    "model": "claude-sonnet-4-20250514",
    "input": "list files in the project",
    "stream": false
  }'
GET/health

健康检查接口。

curl http://localhost:8002/health

Vision / OCR

  • VISION_MODE=ocr：本地 Tesseract + gosseract OCR
  • VISION_MODE=api：外接 OpenAI 兼容视觉模型接口
  • 图片会在发给 Cursor Web 之前先被提取为文本上下文
VISION_ENABLED=true \
VISION_MODE=ocr \
VISION_LANGUAGES=eng,chi_sim \
./cursor2api-go
